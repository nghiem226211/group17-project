Tên dự án: User Management MERN Stack (Nhóm 17).

Công nghệ: Node.js, Express, MongoDB/Mongoose, React, JWT, bcryptjs.

Hướng dẫn chạy: (Gồm 3 bước)

npm install

Tạo file .env (với MONGO_URI và JWT_SECRET).

npm run dev (cho Backend) và cd frontend && npm install && npm start (cho Frontend).
